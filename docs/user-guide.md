# Driver Logbook v3 - ユーザーガイド

## 🚀 はじめに

**Driver Logbook v3** は、委託軽貨物ドライバーの日報管理を効率化する Web アプリケーションです。
日々の稼働記録を簡単に入力・管理し、月次レポートの作成を自動化します。

---

## 📱 アクセス方法

**本番環境**: https://driverlogbook-seven.vercel.app
**推奨ブラウザ**: Chrome, Safari, Edge (最新版)
**対応端末**: スマートフォン、タブレット、デスクトップ

---

## 🔑 アカウント作成・ログイン

### 1. **新規登録**

1. トップページで「新規登録」をクリック
2. メールアドレスとパスワード（6 文字以上）を入力
3. 「アカウント作成」ボタンをクリック
4. **重要**: 登録したメールアドレスに確認メールが届きます
5. メール内のリンクをクリックしてアカウントを有効化

### 2. **ログイン**

1. メールアドレスとパスワードを入力
2. 「ログイン」ボタンをクリック
3. ダッシュボードに自動的にリダイレクトされます

### 3. **ログアウト**

- 画面右上のユーザーメニューから「ログアウト」をクリック

---

## 📊 ダッシュボード

ログイン後に表示される画面で、今月の稼働状況を一目で確認できます。

### 表示される情報

- **📅 稼働日数**: 今月働いた日数
- **🛣️ 総走行距離**: 今月の合計走行距離（km）
- **📦 総配送件数**: 今月の合計配送件数
- **💰 高速料金合計**: 今月の高速道路利用料金合計
- **📈 平均値**: 1 日あたりの平均距離・配送件数

### クイックアクション

- **「新しい日報を作成」**: 今日の稼働記録入力
- **「日報一覧」**: 過去の日報確認・編集
- **「月次レポート」**: 月別データとエクスポート機能

---

## 📝 日報の作成

### 1. **基本操作**

1. ダッシュボードから「新しい日報を作成」をクリック
2. 必要な情報を入力
3. 「保存」ボタンで記録完了

### 2. **入力項目の詳細**

#### 📅 **日付**

- デフォルトで今日の日付が設定されます
- 必要に応じて過去の日付も選択可能

#### ✅ **稼働チェック**

- **ON**: 今日働いた場合
- **OFF**: 休日・非稼働日の場合

#### ⏰ **時間入力** (稼働日のみ)

- **開始時間**: 業務開始時刻
- **終了時間**: 業務終了時刻
- **「現在」ボタン**: ワンクリックで現在時刻を入力

#### 🚗 **メーター入力** (稼働日のみ)

- **開始メーター**: 業務開始時のオドメーター値
- **終了メーター**: 業務終了時のオドメーター値
- **自動計算**: 走行距離が自動的に計算されます
- **メーター巻き戻り対応**: 999,999km を超えた場合も正確に計算

#### 📦 **配送・経費情報** (稼働日のみ)

- **配送件数**: 1 日の配送件数を入力
- **高速料金**: 高速道路利用料金を入力（円）

#### 📋 **備考**

- その他のメモや特記事項を自由に記入

### 3. **便利機能**

#### **前回メーター値の自動取得**

- 開始メーター欄に、前回の終了メーター値が自動的に表示されます
- 連続した記録入力が簡単になります

#### **リアルタイム距離計算**

- メーター値を入力すると、走行距離が自動計算されて表示されます

#### **バリデーション機能**

- 必須項目の未入力
- 不正な値（負の数値など）
- 論理的におかしい値（終了＜開始など）
  などをリアルタイムでチェックし、エラーメッセージを表示します

## 📋 日報の管理

### 1. **日報一覧の確認**

1. ダッシュボードから「日報一覧」をクリック
2. 過去の日報が新しい順で表示されます

### 2. **検索・フィルター機能**

- **日付範囲**: 開始日〜終了日で絞り込み
- **稼働状況**: 稼働日のみ / 非稼働日のみ / 全て
- **キーワード検索**: 備考欄の内容で検索

### 3. **日報の編集**

1. 一覧から編集したい日報の「編集」ボタンをクリック
2. 内容を修正して「更新」ボタンをクリック

### 4. **日報の削除**

1. 一覧から削除したい日報の「削除」ボタンをクリック
2. 確認ダイアログで「削除」を選択
3. **注意**: 削除したデータは復元できません

---

## 📊 月次レポート・エクスポート

### 1. **月次レポートの表示**

1. ダッシュボードから「月次レポート」をクリック
2. 年月を選択（プルダウンメニュー）
3. 該当月の統計情報が自動計算されて表示

### 2. **表示される統計情報**

- **稼働日数**: 該当月に働いた日数
- **総走行距離**: 該当月の合計走行距離
- **総配送件数**: 該当月の合計配送件数
- **総高速料金**: 該当月の高速道路利用料金合計
- **平均値**: 1 日あたりの平均距離・配送件数
- **日別詳細**: カレンダー形式での日別データ表示

### 3. **PDF エクスポート**

1. 月次レポート画面で「PDF エクスポート」ボタンをクリック
2. 美しくフォーマットされた月次レポート PDF が自動生成
3. ブラウザの標準機能で保存・印刷が可能

**PDF の内容**:

- 月次統計サマリー
- 日別詳細テーブル
- 会社情報・期間情報
- 日本語フォント対応

### 4. **CSV エクスポート（3 つの形式）**

#### **基本形式**

- 日常的な確認用
- 日付、稼働状況、距離、配送件数など主要項目

#### **詳細形式**

- 完全なデータエクスポート
- 全ての入力項目と計算結果を含む
- 作業時間の計算結果も含む

#### **経理用形式**

- 会計ソフトへの取り込み用
- 数値データを中心とした構成
- 経費計算に特化した項目配置

**CSV の特徴**:

- UTF-8 BOM 対応（Excel で文字化けしない）
- 日本語項目名
- 標準的な CSV 形式

## ⚠️ 注意事項・制限事項

### セキュリティ

- **個人データ保護**: 入力データは暗号化されてデータベースに保存
- **認証必須**: 他のユーザーのデータにはアクセス不可
- **セッション管理**: 一定時間でログアウトされます

### データの制限

- **日付の重複**: 同じ日付の日報は 1 つまで
- **数値の範囲**: メーター値は 0〜999,999km
- **文字数制限**: 備考欄は 500 文字まで

### 動作環境

- **インターネット接続必須**: オフラインでは動作しません
- **JavaScript 有効**: ブラウザで JavaScript を有効にしてください
- **Cookie サポート**: 認証に Cookie を使用します

## 🆘 よくある質問・トラブルシューティング

### Q1: ログインできません

**A1**: 以下を確認してください

- メールアドレスの確認メールからアカウント有効化済みか
- パスワードが正しいか（6 文字以上）
- ブラウザの Cookie が有効か

### Q2: データが表示されません

**A2**: 以下を確認してください

- インターネット接続が安定しているか
- ブラウザをリフレッシュしてみる
- 別のブラウザで試してみる

### Q3: 距離計算が正しくない

**A3**: 以下を確認してください

- 開始メーター ＜ 終了メーター になっているか
- メーター値が現実的な範囲内か
- メーターが 999,999km を超えた場合は自動で巻き戻り計算されます

### Q4: エクスポートファイルが開けません

**A4**: 以下を試してください

- **PDF**: 最新の PDF リーダーを使用
- **CSV**: Excel やスプレッドシートアプリで開く
- 文字化けする場合は「UTF-8」でファイルを開く

### Q5: スマートフォンで使いにくい

**A5**: 以下を試してください

- ブラウザを最新版に更新
- 画面を横向きにして操作
- ズーム機能を使って拡大表示
